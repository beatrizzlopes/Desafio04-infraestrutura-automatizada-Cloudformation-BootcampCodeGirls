# Anotações

### A importância de utilizar automação da infraestrutura

A automação da infraestrutura permite criar, configurar e gerenciar recursos de forma automática, reduzindo erros manuais, garantindo consistência e acelerando o provisionamento de ambientes na nuvem.

### O que é AWS CloudFormation

O AWS CloudFormation é um serviço que permite modelar e automatizar a criação e configuração de recursos na AWS. Com ele, podemos definir, em modelos (templates), recursos como instâncias EC2, bancos de dados RDS e muito mais, garantindo que sejam provisionados de forma consistente e automatizada

### Como criar uma infraestrutura com AWS CloudFormation

1. Criar o template – Defina os recursos que você precisa em um arquivo JSON ou YAML.

2. Aplicar no CloudFormation – Suba o template no serviço AWS CloudFormation.

3. Gerar a stack – O CloudFormation lê o template e cria a stack, provisionando e configurando automaticamente todos os recursos definidos.
