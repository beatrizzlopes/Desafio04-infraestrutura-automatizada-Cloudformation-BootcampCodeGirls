# Anotações

### O que é AWS CloudFormation

O AWS CloudFormation é um serviço que permite modelar e automatizar a criação e configuração de recursos na AWS. Com ele, podemos definir, em modelos (templates), recursos como instâncias EC2, bancos de dados RDS e muito mais, garantindo que sejam provisionados de forma consistente e automatizada
